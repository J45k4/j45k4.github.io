Dijkstra's Algorithm

How it Works

Dijkstra's Algorithm is used to find the shortest path in a weighted graph from a starting node to all other nodes.

  • Initialize distances to all nodes as infinity, except the starting node which is 0.
  • Use a priority queue to keep track of nodes with the shortest distance.
  • Pick the node with the smallest known distance and update its neighbors.
  • Repeat until all nodes are visited.

Applications

  • Route planning in GPS.
  • Network routing protocols.
  • Shortest path problems in robotics.

Code Example

	  
import heapq

def dijkstra(graph, start):
	distances = {node: float('inf') for node in graph}
	distances[start] = 0
	queue = [(0, start)]

	while queue:
	current_distance, current_node = heapq.heappop(queue)

	if current_distance > distances[current_node]:
		continue

	for neighbor, weight in graph[current_node].items():
		distance = current_distance + weight

		if distance < distances[neighbor]:
		distances[neighbor] = distance
		heapq.heappush(queue, (distance, neighbor))

	return distances
