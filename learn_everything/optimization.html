Optimization

Linear Programming

Linear programming is used to find the best outcome in a mathematical model whose requirements are represented by linear relationships.

Examples:

  • Simplex Method
  • Graphical Method

Convex Optimization

In convex optimization, the set over which the optimization is performed is a convex set and the function to optimize is a convex function.

Examples:

  • Convex Functions
  • KKT Conditions

Dynamic Programming

Dynamic programming is used for optimizing recursive problems by storing past results to avoid redundant calculations.

Examples:

  • Fibonacci Sequence
  • Shortest Path
