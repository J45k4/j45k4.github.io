Data Structures

Arrays

Simple, fixed-size collections of elements.

Examples:

  • Array Declaration
    • int[] arr = new int[5];
  • Accessing Elements
    • arr[0] = 1;

Linked Lists

Collections of elements, called nodes, that are connected.

Examples:

  • Singly Linked List
    • class Node {
  int data;
  Node next;
}
  • Doubly Linked List
    • class Node {
  int data;
  Node next;
  Node prev;
}
